## **Raw Data**
The raw Used Cars Dataset has been omitted from this GitHub due to its file size.\
The dataset can be obtained from Kaggle [ https://www.google.com/url?q=https%3A%2F%2Fwww.kaggle.com%2Fdatasets%2Faustinreese%2Fcraigslist-carstrucks-data%2Fdata ]\
Ensure to include the raw dataset in this directory and update the file paths accordingly if you plan to perform Data Cleaning and Preprocessing on the dataset.
